# Instalação

Use o script adequado para seu sistema.

## Windows

```powershell
powershell -ExecutionPolicy Bypass -File .\install-nicy.ps1
```

## Linux / macOS / Android (Termux)

```bash
bash install-nicy.sh
```

Com repositórios privados:

```bash
GITHUB_TOKEN=seu_token bash install-nicy.sh
```
