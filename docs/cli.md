# CLI Guide

Use this page as practical command usage companion.

## Main commands

```bash
nicy run main.luau
nicy eval "print(runtime.version)"
nicy compile main.luau
nicy runtime-version
```

## Notes

- keep runtime binary reachable in executable dir or PATH
- use `runtime-version` as first health check

For strict command contracts, see [/reference/cli](/reference/cli).
